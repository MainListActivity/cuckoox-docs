# 管理人员登录逻辑说明

管理人员账号有两种创建途径：
> 1. 从业人员登录App端，选择自己的身份（公、检、法、律）（暂时不开放，计划增加从业人员身份认证功能，选择身份后需要提供身份证明材料）
> 2. 已有账号的从业人员在[账号管理页面](/docs/dev/account/account-manage) 通过添加账号到自己的案件/组织
> 计划增加从业人员同步功能，可以对接第三方账号LDAP

管理人员的账号在创建案件时绑定到案件，这里的管理人员指的就是案件负责人/案件管理人员；而用户登录后可以查看所有关联的案件
（组织无法无法直接关联案件，每个案件都需要由一个具体的人创建，而创建人所属的组织会自动拥有案件的查看权限，每个组织会自动生成一个监控账号，用于查看所有案件监控大屏），
选择对应的案件后进入工作台，案件的创建人可以在工作台查看单个案件的进度。





:::note
___注意:___ 从业人员关联到具体的案件即可称为管理人员
:::
